# render-engine

A simple renderer.

To build and run on *nix systems:

```
mkdir cmake-build
cd cmake-build
cmake ..
make -j16
./engine
```

To build on windows, see:

[build for visual studio on windows](./doc/How-to-build-under-M$-Windows.md)
